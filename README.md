# Math-test
\*an epic intro\*
A new game designed by meh. It's an easy math game. In math problems will appear only numbers from ZERO to NINE. So, it is veeeeery eeeeeeeeeeaaaaaaaaaassssssssssssyyyyyyyyy! Every stage (there is 9 stages) has 3 levels, every level has 5 math questions. In each level u have some bonus points. Every next stage costs 50% more than prevous. Second Stage costs 500 points :D
